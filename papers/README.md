## Notes

- **#5 substitution:** Palli & Melchiorri 2006 ICRA (the canonical Bowden-sheath
  modeling paper) is paywalled, so I dropped in **Borghesan, Palli, Melchiorri 2010
  ICRA**, *Design of tendon-driven robotic fingers: Modeling and control issues* — a
  free follow-up from the same lab covering similar friction/hysteresis/compliance
  ground. Useful as a placeholder until you can grab the 2006 one via Auburn proxy.
  Read both if possible; the 2010 paper is more design-oriented, the 2006 paper is
  more fundamental on the transmission model itself.
- **#8 is 24 MB** because the Science Advances open-access PDF bundles figures and
  supplementary materials. Scan-only paper anyway per the roadmap.
- **LEAP v2 (#2b)** is short — workshop paper, mostly delta from v1. Treat as a
  30-min skim per the roadmap.
- All filenames follow the pattern `<number>-<first-author>-<year>-<short-tag>.pdf`
  so new additions sort correctly.
